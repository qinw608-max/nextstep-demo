# 下一步 — 产品规格

状态：v3 — 比赛展示 Demo 完成，产品开发待启动
日期：2026-06-24

## 1. 产品定位

**下一步** 是一款外挂插件——不替换你的 Agent，不改变对话习惯，嵌在对话窗口旁边。

解决长对话执行中的上下文丢失：任务、资料、提问、Agent 回复和完成结果都能回到对应节点。做完了一步步收归，最终沉淀成可复用模板。

**差异化**：不是多 Agent 编排器，不是普通看板。核心是执行过程的节点化收束——选中节点里任意一段文字直接向 Agent 提问，回复写回原节点。

## 2. 核心功能（8个）

| # | 功能 | 说明 |
|---|---|---|
| 01 | 全局资产扫描 | 扫描电脑文件，按板块自动归类，一眼看到你有什么 |
| 02 | 双向同步 | 对话→画布自动更新，画布→对话也能回到对话继续 |
| 03 | 节点即画布 | 节点是资料容器，双击进入节点内部画布空间，可挂文件/图片/文件夹/注释，多层嵌套 |
| 04 | Agent 实时编辑 | agent 在文件里写内容时，画布上文件节点实时更新 |
| 05 | 批注工具 | 任意位置贴批注，虚线连接节点 |
| 06 | Agent 协作 | 选中节点内容→提问→agent回复→写回原节点 |
| 07 | 抽屉收归 | 完成的节点折叠，画布保持清爽 |
| 08 | 模板沉淀 | 已完成节点沉淀为模板，下次直接调用 |

## 3. 节点设计

节点不是任务步骤，是资料容器 + 工作空间：

- **双击进入节点**：画布相机平滑缩放进入节点内部空间，不是弹窗
- **节点内部是画布**：可以放文件（显示完整内容）、图片（完整大图）、文字段落、注释
- **多层嵌套**：父节点 → 子节点 → 子节点的子节点，无限层级
- **面包屑导航**：进入节点后显示路径，点击任一层级返回
- **精细提问**：节点内任意文字可选中→弹出"向 agent 提问"→回复写回原节点
- **收归**：完成后折叠+标记完成，一步步收归，最终变成一个文件夹

## 4. 痛点（4条）

1. 对话写到第 30 轮，你已经开始往上翻了
2. 资料散落各处，agent 看不到你有什么
3. 能在对话里让 agent 帮忙，但不能在画布上一起协作
4. 思路是树状的，工具都是扁平的

## 5. 应用场景

小说创作、报税整理、参赛项目、产品开发、学术研究、学习笔记——任何需要和 agent 深度协作的任务。
