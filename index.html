给 agent 配一块画布

对话窗口管执行，画布管全局，两者双向同步

— 你遇到过这些问题吗 —

对话很强，但你看不见全局

01

对话写到第 30 轮，你已经开始往上翻

主角的背景设定是第几轮定的？那个方案选了 A 还是 B？伏笔埋了没？对话是流，流过去就没了，你只能往上翻。写小说翻几十轮找角色设定，做报税翻对话找哪个扣除项确认了，参加比赛翻对话找技术方案到底定了没。你不知道任务做到哪了、哪步漏了、逻辑链清不清。

02

资料散落各处，agent 看不到你有什么

参赛资料在 A 文件夹，小说素材在 B 文件夹，报税凭证在 C 文件夹。你和 agent 对话时，它不知道你电脑上已经有什么。每次都要手动说"参考一下那个文件"——但你自己都不一定记得那个文件在哪。没有资产的可视化管理，你和 agent 都在盲人摸象。

03

能让 agent 帮忙，但不能在画布上一起协作

现在 agent 的交互只有对话一条路。选中一段内容想追问？得复制粘贴回对话框。想让 agent 改某个具体环节？得用文字描述清楚是哪个环节。在画布上点开节点、选中内容、直接提问——这种"看着同一张图协作"的体验，目前没有。

04

思路是树状的，工具都是扁平的

一个角色设定下挂着外貌、性格、背景故事，背景故事下又挂着童年、关键事件。一个需求下拆出方案、设计、实现，实现下又分前端、后端。思考天然是多层嵌套的树状结构，但现在的对话是线性的，任务列表是扁平的。没有一个地方能让你把思路像笔记一样展开、挂资料、加注释、多层嵌套。

下一步 是一款插件——嵌在 agent 旁边，对话窗口不动，外挂一个画布。节点是资料容器：可以挂图片、文件、文件夹，可以加注释，可以多层嵌套。你在画布上和 agent 看着同一张图协作，做完了一步步收归，最终变成一个文件夹。

— 适用场景 —

任何需要和 agent 深度协作的任务

✦

小说创作

角色卡、世界观、伏笔追踪、章节大纲，多层嵌套管理。几十轮对话不翻，画布上一目了然。

§

报税整理

收入凭证、专项扣除、申报流程，按模块归类。和 agent 确认每一步，画布上看到进度和缺口。

◆

参赛项目

需求分析、技术方案、界面设计、提交。agent 拆分任务到画布，你逐步完成，沉淀成模板。

▸

产品开发

需求拆解、前后端任务、设计稿、API 文档，全部挂到节点上。agent 和你看着同一张图协作。

✎

学术研究

文献综述、研究方法、数据分析、论文结构。资料和笔记分层挂载，不丢失上下文。

⬡

学习笔记

课程章节、知识点、代码片段、练习项目。学完一个收起一个，最终沉淀成知识体系。

— 体验 Demo —

8 个可交互的 Demo

01 扫描你的资产 看见全局
01

全局资产扫描

点击中心按钮启动雷达扫描，文件按板块飞入画布。每个板块显示任务数、完成数、资产数。点击板块卡片展开文件列表，新建模块。

LIVE
02

双向同步

左侧 agent 对话自动播放，右侧画布实时更新节点。点击节点"完成"按钮，对话窗口出现 agent 确认。选中节点内容可以提问，agent 回复后画布更新。

LIVE
02 画布和对话双向同步 双向协作
03

节点是资料容器

节点不只是任务步骤。可以挂文件、图片、文件夹，可以加注释，可以多层嵌套展开——父节点到子节点到子节点的子节点。双击编辑正文，点标题展开折叠子节点，完成后收归成文件夹。

LIVE
04

Agent 编辑时实时看见

agent 在文件里写内容时，你不用翻对话——画布上的文件节点实时更新。agent 每写一句，节点上的预览同步增长，有闪烁提示。节点顶部显示"编辑中"状态。这就是"知其所以然"。

LIVE
05

批注工具

切到批注模式，点击画布任意位置贴批注。可编辑、可拖拽、可删除。虚线自动连接到最近节点，不丢失上下文。

LIVE
06

Agent协作

选中节点内容→弹出提问框→agent 回复→节点自动更新。也可以让 agent 直接在画布新建节点，看着同一张图协作。

LIVE
03 做完变模板，下次直接调用 沉淀复用
07

抽屉收归

点击"收起"按钮，节点折叠成一行标题。完成节点时自动收起。可以全部收起或全部展开，画布保持清爽不丢信息。

LIVE
08

模板沉淀

已完成节点汇聚成模板卡片，存入模板库。点击模板→确认应用→展开成新任务节点飞入画布。流程从真实执行中长出来，下次直接调用。
